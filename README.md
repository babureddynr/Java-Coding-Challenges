# Java Logical Programs for Interview Coding

Welcome to the **Java Logical Programs** repository! 🚀 This collection contains a variety of logical problems frequently asked in technical interviews, designed to help you sharpen your coding skills and problem-solving abilities.

## 📂 Folder Structure

```bash
├── ArrayPrograms
├── StringPrograms
├── SortingAlgorithms
├── NumberPrograms
├── RecursionPrograms
├── PatternPrograms
```

Each folder contains specific types of Java problems, categorized by the type of problem or algorithm.

## 📝 List of Topics Covered

- **Array Programs**:  
  - Find the largest element in an array
  - Reverse an array
  - Merge two sorted arrays
  - Find the missing number in an array

- **String Programs**:  
  - Check for palindrome strings
  - Reverse a string without using additional memory
  - Count vowels and consonants in a string
  - Find duplicates in a string

- **Sorting Algorithms**:  
  - Bubble sort
  - Merge sort
  - Quick sort
  - Selection sort

- **Number Programs**:  
  - Check for prime numbers
  - Fibonacci series
  - Factorial of a number
  - Sum of digits of a number

- **Recursion Programs**:  
  - Calculate the factorial of a number using recursion
  - Fibonacci series using recursion
  - Solve the Tower of Hanoi problem
  - Recursive string reversal

- **Pattern Programs**:  
  - Print pyramid patterns
  - Print diamond patterns
  - Print number triangles
  - Print Pascal’s triangle

- **Miscellaneous Programs**:  
  - Swap two numbers without using a temporary variable
  - Find the GCD of two numbers
  - LCM of two numbers
  - Find Armstrong numbers

## 🔧 How to Run the Programs

1. **Clone the repository:**
   ```bash
   git clone https://github.com/babureddynr/Java-Coding-Challenges.git
   ```

2. **Open the project in your preferred IDE (Eclipse, IntelliJ, etc.)**.

3. **Navigate to the desired program** in the folder structure and run it.

4. Modify or play with the logic to enhance your understanding.
